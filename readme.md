# Food Banner Website

A visually appealing and responsive food banner landing page designed to showcase a food delivery service's reach and impact.

## Features

- **Interactive Parallax Effect**: Food items and the background subtly react to mouse movement, creating a dynamic 3D depth effect.
- **Smooth Animations**:
    - **Entrance Animations**: Content fades and scales in gracefully on page load.
    - **Floating Effect**: Food items have continuous, subtle bobbing animations to feel "alive."
    - **Hover States**: Interactive scaling and rotation on food items and info cards.
- **Hero Section**: Engaging headline and descriptive text highlighting the mission of providing better food.
- **Visual Food Elements**: Floating food items (burgers, pizzas, momos, etc.) for a dynamic and appetizing feel.
- **Statistics Section**: Highlights key metrics with a blurred backdrop-filter for a modern "glassmorphism" look.
- **Responsive Design**: Fully optimized for various screen sizes, including:
    - Mobile devices (up to 768px)
    - Tablets and Laptops (769px to 1200px)
    - Desktop monitors (1201px and above)
- **Modern Typography**: Uses the "Inter" font family with bold, high-contrast styles.

## Technologies Used

- **HTML5**: Semantic structure of the webpage.
- **CSS3**: Custom styling, including Flexbox, absolute positioning, and Media Queries for responsiveness.
- **Google Fonts**: Integration of the "Inter" font.

## How to Run

1. Clone or download the repository to your local machine.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge).
3. Experience the responsive design by resizing your browser window or using developer tools to simulate different devices.
